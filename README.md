# TajMarket - Электронная торговая площадка в Таджикистане

**TajMarket** — это веб-платформа для интернет-магазина и электронной коммерции в Таджикистане, разработанная на языке Python.

## Технологии (Stack):
* **Backend:** Python, Django, Django REST Framework (DRF)
* **Database:** SQLite / PostgreSQL
* **Features:** Фильтрация товаров по районам (districts) и категориям.

## О проекте
Проект разработан для автоматизации онлайн-торговли, создания удобного API для мобильных приложений и веб-сайтов.
